## default Mode

### File
- New: `CmdOrCtrl+N`
- Open: `CmdOrCtrl+O`
- Save: `CmdOrCtrl+S`

### Edit
- Insert: `I`
- Copy: `CmdOrCtrl+C`
- Paste: `CmdOrCtrl+V`
- Undo: `CmdOrCtrl+Z`
- Delete: `Backspace`
- Move Up: `Up`
- Move Down: `Down`
- Move Left: `Left`
- Move Right: `Right`
- Deselect: `Esc`

### Stroke
- Line: `A`
- Arc: `S`
- Arc Rev: `D`
- Bezier: `F`
- Connect: `Z`

### Effect
- Thicker: `}`
- Thinner: `{`
- Thicker +5: `]`
- Thinner -5: `[`
- Linecap: `Y`
- Mirror: `Space`
- Fill: `G`

### View
- Tools: `U`
- Grid: `H`
- Control Points: `J`
- Expert Mode: `:`
